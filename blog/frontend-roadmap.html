前端能力模型与成长路径

1. 核心技术能力

  • 基础牢固：HTML 语义化、CSS 布局、JavaScript / TypeScript 语言特性
  • 框架掌握：至少熟练 Vue 与 React 之一，并理解响应式原理
  • 工程体系：模块化、构建优化、质量保障（Lint、Test、CI/CD）

2. 业务理解与抽象

  • 透过需求看到流程、角色、指标，抽象出可复用的模型
  • 善用状态图、时序图描述复杂度
  • 在“快迭代”与“高质量”之间找到平衡

3. 深度方向

方向关键能力业务价值
体验设计动效、可访问性、设计协作转化率、品牌感知
可视化图表、地图、3D、动画决策效率、差异化
工程效率CLI、脚手架、监控平台人效、稳定性

4. 成长建议

  1. 复盘驱动：每个项目输出“技术复盘 + 业务复盘”
  2. 持续分享：写博客、内部分享、开源贡献，巩固认知
  3. 跨界沟通：多与设计、后端、产品合作，理解全链路
