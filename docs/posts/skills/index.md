# 技能

## 技能概览

- 技术栈：Vue2/3、TypeScript、Uniapp、Pinia/Vuex、Vue Router
- 工程化：Vite/Webpack、Git 流程、自动化脚本（Node.js）、Nginx 部署
- 可视化：ECharts、秋云，常用 UI 组件库（Element/Vant/uni-ui）

## 学习与规划

- 持续迭代工程化工具链与组件库实践
- 强化可视化与交互体验的设计与实现

## 近期目标

- 完成一个多端适配的图表库封装（ECharts + 统一主题）
- 将请求/状态/路由整合为可复用的 Starter 模板
- 编写 3 篇关于性能优化与可视化设计的技术文章

## 路线图

1. 提炼通用组件（表单、列表、图表卡片）
2. 建立性能监控与优化基线（Lazy、Preload、分包）
3. 封装脚手架工具，提升项目搭建与部署效率
